
1. Haga clic en proyecto de tienda Windows hello, haga clic en **establecer como proyecto de inicio**, a continuación, presione Hola F5 toorun clave Hola aplicación la tienda Windows.
   
    Una vez iniciada la aplicación hello, dispositivo Hola se registra para las notificaciones de inserción.
2. Detener la aplicación de la tienda de Windows hello y repita el paso anterior de hello para la aplicación de la tienda de Windows Phone de Hola.
   
    En este punto, ambos dispositivos son las notificaciones de inserción de tooreceive registrados.
3. Vuelva a ejecutar la aplicación de la tienda de Windows hello y escriba texto en **insertar un TodoItem**y, a continuación, haga clic en **guardar**.
   
       Note that after hello insert completes, both hello Windows Store and hello Windows Phone apps receive a push notification from WNS. hello notification is displayed on Windows Phone even when hello app isn't running.
