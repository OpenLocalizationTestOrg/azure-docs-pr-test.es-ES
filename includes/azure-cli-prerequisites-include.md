## <a name="prerequisite-install-the-azure-cli"></a>Requisito previo: instalar la CLI de Azure
Para seguir los pasos de este artículo, es preciso [instalar la interfaz de la línea de comandos de Azure para Mac, Linux y Windows (CLI de Azure)](../articles/cli-install-nodejs.md) e [iniciar sesión en Azure](/cli/azure/authenticate-azure-cli). 

> [!NOTE]
> Si no tiene una cuenta de Azure, necesitará una. Regístrese para [obtener una prueba gratuita aquí](../articles/active-directory/sign-up-organization.md). Además, para seguir el tutorial por completo necesita disponer de [jq](https://stedolan.github.io/jq/) o de alguna otra herramienta de análisis o biblioteca de JSON instalada.
