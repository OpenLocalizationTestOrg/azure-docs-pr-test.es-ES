Un equilibrador de carga interno (ILB) de Azure proporciona un equilibrio de carga de red entre las máquinas virtuales que residen dentro de un servicio en la nube o de una red virtual con un ámbito regional.

Para obtener información sobre el uso de Hola y la configuración de redes virtuales con un ámbito regional, consulte [redes virtuales regionales](../articles/virtual-network/virtual-networks-migrate-to-regional-vnet.md). Las redes virtuales existentes que se han configurado para un grupo de afinidad no pueden usar ILB.
