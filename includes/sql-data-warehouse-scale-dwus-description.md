Puede aumentar o reducir el proceso ajustando Unidades de almacenamiento de datos (DWU). El valor predeterminado es 400 DWU. Los recursos de proceso aumentan linealmente a medida que lo hacen DWU. Por ejemplo, si se cambian DWU de 400 a 800 se doblarán los recursos de proceso. 

Para obtener una lista de los valores de DWU actuales, consulte el objetivo de nivel de servicio en [Service capacity limits](../articles/sql-data-warehouse/sql-data-warehouse-service-capacity-limits.md)(Límites de capacidad de servicio).
