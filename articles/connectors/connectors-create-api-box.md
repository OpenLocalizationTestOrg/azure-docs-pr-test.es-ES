# <a name="get-started-with-hello-box-connector"></a>Empezar a trabajar con el conector de la caja de Hola
Conectar el cuadro de herramientas y crear archivos, elimine los archivos y mucho más. Con Box, puede hacer lo siguiente:

* Compilar el flujo de negocios basado en datos de hello que obtendrá de cuadro. 
* Usar desencadenadores para cuando se crea o actualiza un archivo.
* Usar acciones que copian o eliminan archivos, entre otras muchas cosas. Estas acciones obtención una respuesta y, a continuación, asegúrese de salida de hello disponibles para las demás acciones. Por ejemplo, cuando se modifique un archivo en Box, puede tomar ese archivo y enviarlo por correo electrónico mediante Office 365.

Puede empezar creando una aplicación lógica ahora. Para ello, consulte [Creación de una aplicación lógica](../logic-apps/logic-apps-create-a-logic-app.md).
