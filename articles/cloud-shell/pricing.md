# <a name="pricing"></a>Precios

## <a name="compute-cost"></a>Costo de proceso
Shell de nube de Azure se ejecuta en un equipo que se proporciona de forma gratuita, pero requiere una toouse de recurso compartido de archivo adjunto de Azure. 

## <a name="storage-cost"></a>Costo del almacenamiento
Shell de nube crea una imagen predeterminada de 5GB en su toopersist de recurso compartido de archivo su directorio $HOME. Los recursos compartidos de archivos de Azure conllevan costos de forma regular.

Haga clic [aquí para obtener detalles sobre los costos de Azure Files](https://azure.microsoft.com/pricing/details/storage/files/).
