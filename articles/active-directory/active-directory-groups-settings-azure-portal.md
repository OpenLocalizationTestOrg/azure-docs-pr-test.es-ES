# <a name="manage-hello-settings-for-a-group-in-azure-active-directory"></a>Administrar la configuración de Hola para un grupo en Azure Active Directory
Este artículo explica cómo toochange Hola configuración para un grupo en Azure Active Directory (Azure AD).

## <a name="how-do-i-find-and-change-hello-settings"></a>¿Cómo buscar y cambiar la configuración de hello?
1. Inicie sesión en toohello [centro de administración de Azure AD](https://aad.portal.azure.com) con una cuenta que sea un administrador global para el directorio de Hola.
2. Seleccione **más servicios**, escriba **usuarios y grupos** en Hola cuadro de texto y, a continuación, seleccione **ENTRAR**.

   ![Apertura de la hoja de usuarios y grupos](./media/active-directory-groups-settings-azure-portal/search-user-management.png)
3. En hello **usuarios y grupos** hoja, seleccione **todos los grupos de**.

   ![Hola de apertura todos los grupos de hoja](./media/active-directory-groups-settings-azure-portal/view-groups-blade.png)
4. En hello **a los usuarios y grupos: todos los grupos de** hoja, seleccione un grupo.
5. En hello **grupo - *groupname***  hoja, seleccione **propiedades**.

   ![Hoja de propiedades de apertura Hola](./media/active-directory-groups-settings-azure-portal/select-group-properties.png)
6. Cuando termine de cambiar las propiedades de grupo de hello, seleccione **guardar**.    

   ![Guardado de los cambios de propiedades](./media/active-directory-groups-settings-azure-portal/save-group-properties.png)
