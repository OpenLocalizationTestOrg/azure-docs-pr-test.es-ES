* <span data-ttu-id="6bf56-111">**Seguridad mejorada** : los datos se cifran automáticamente antes de salir del dispositivo Windows 10 del usuario mediante Azure Rights Management (Azure RMS) y los datos permanecen cifrados almacenados en la nube.</span><span class="sxs-lookup"><span data-stu-id="6bf56-111">**Enhanced security** – Data is automatically encrypted before leaving the user’s Windows 10 device by using Azure Rights Management (Azure RMS), and data stays encrypted at rest in the cloud.</span></span> <span data-ttu-id="6bf56-112">Todo el contenido permanece cifrado en la nube, excepto los espacios de nombres, como nombres de configuración y nombres de aplicación de Windows.</span><span class="sxs-lookup"><span data-stu-id="6bf56-112">All content stays encrypted at rest in the cloud, except for the namespaces, like settings names and Windows app names.</span></span>
