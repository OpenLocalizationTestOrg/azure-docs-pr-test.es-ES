# <a name="getting-support-for-b2b-collaboration"></a>Obtención de soporte para colaboración B2B

¿Ha leído la documentación hello, hecho cosas derecho hello, pero todavía no se puede obtener algo toowork? Abra una incidencia de soporte técnico (requiere un plan de soporte técnico):

1. En [Hola portal de Azure](https://portal.azure.com), navegar por toohello ayuda y la hoja de soporte técnico y seleccione nueva solicitud de soporte técnico:
  - Tipo de problema: Técnico
  - Suscripción: elija la suscripción afectada
    - Servicio: Active Directory
    - Plan de soporte técnico: elija el plan de soporte técnico pertinente

  ![Nueva solicitud de soporte técnico](media/active-directory-b2b-support/new-support-request.png)

2. Describa el problema:
  - Elija gravedad adecuado de Hola que refleja necesita.
  - Elija el tipo de problema Administración de usuarios y grupos
  - Elija la categoría Agregando usuarios (B2B)
  - Incluya cualquier mensaje de error, como CorrelationID, los usuarios afectados y así sucesivamente.

  ![Incidencia de soporte técnico, descripción del problema](media/active-directory-b2b-support/problem-description.png)

3. Para una toocontact representante de soporte técnico para solucionar problemas adicionales, agregar la información de contacto.
