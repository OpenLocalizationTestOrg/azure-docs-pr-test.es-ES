## <a name="suitable-scenarios-for-the-oauth2-implicit-grant"></a><span data-ttu-id="04bd1-121">Escenarios adecuados para la concesión implícita de OAuth2</span><span class="sxs-lookup"><span data-stu-id="04bd1-121">Suitable scenarios for the OAuth2 implicit grant</span></span>
