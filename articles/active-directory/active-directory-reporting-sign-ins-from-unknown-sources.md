# <a name="sign-ins-from-unknown-sources"></a>Inicios de sesión desde orígenes desconocidos
Este informe indica los usuarios que han iniciado sesión correctamente en el directorio de tooyour mientras tenían asignada a un cliente de dirección IP que se ha identificado por Microsoft como una dirección IP de proxy anónima (por ejemplo, una dirección IP Tor). Estos servidores proxy son a menudo utilizado por usuarios que quieran toohide dirección IP de su equipo y puede utilizarse de forma malintencionada.

Los resultados de este informe aparecerá el número de Hola de veces que un usuario iniciado sesión correctamente en el directorio tooyour desde esa dirección y la dirección IP del proxy de Hola.
