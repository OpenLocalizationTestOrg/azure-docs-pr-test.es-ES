# <a name="configure-localization"></a>Configuración de la localización
Azure Media Clipper está disponible en 18 idiomas. Para establecer el idioma del widget, debe definir el parámetro `language` durante la inicialización. Pase la cadena de código de idioma deseada desde la lista siguiente:
- Chino (simplificado): zh-hans
- Chino (tradicional): zh-hant
- Checo (cs)
- Neerlandés, Flamenco: nl
- Inglés (en)
- Francés (fr)
- Alemán (de)
- Húngaro (hu)
- Italiano (it)
- Japonés (ja)
- Coreano (ko)
- Polaco (pl)
- Portugués (Brasil): pt-br
- Portugués (Portugal): pt-pt
- Ruso (ru)
- Español (es)
- Sueco (sv)
- Turco (tr)

Para establecer un diccionario de idiomas personalizado o ampliar el predeterminado, debe definir el parámetro `languages` o `extraLanguages`, respectivamente. Pase un diccionario personalizado con el formato JSON siguiente:

```javascript
{
    '{language-code}': {
        '{message-id}': '{message}',
        ...
    }
    ...
}
```
