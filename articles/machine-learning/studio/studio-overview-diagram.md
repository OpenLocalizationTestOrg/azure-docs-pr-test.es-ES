# <a name="overview-diagram-of-azure-machine-learning-studio-capabilities"></a>Diagrama de información general de las funcionalidades de Estudio de aprendizaje automático de Azure
El diagrama **Información general de las funcionalidades de Estudio de aprendizaje automático de Microsoft Azure** proporciona una descripción general de cómo puede usar Estudio de aprendizaje automático para desarrollar un modelo de análisis predictivo y ponerlo en marcha en la nube de Azure.

[Estudio de aprendizaje automático de Azure](https://studio.azureml.net/) dispone de un gran número de algoritmos de aprendizaje automático, junto con módulos que ayudan con la entrada, la salida, la preparación y la visualización de los datos. Mediante estos componentes puede desarrollar un experimento de análisis predictivo, realizar iteraciones sobre él y usarlo para entrenar su modelo.
Luego, con un solo clic puede poner en marcha su modelo en la nube de Azure de forma que se pueda usar para puntuar nuevos datos.

En este diagrama, se muestra cómo encajan todas las piezas juntas.

> [!NOTE]
> Consulte la [hoja de referencia de algoritmos de aprendizaje automático para Estudio de aprendizaje automático de Microsoft Azure](algorithm-cheat-sheet.md) para obtener información adicional sobre cómo revisar y elegir los algoritmos de aprendizaje automático disponibles en Estudio de aprendizaje automático.
