# <a name="get-to-know-your-data-science-virtual-machine"></a>Conozca Data Science Virtual Machine

Data Science Virtual Machine (**DSVM**) incluye una amplia lista de herramientas de ciencia de datos, aprendizaje automático/AI y plataformas de datos. Se han definido en configuraciones que han probado de forma rigurosa científicos de datos y desarrolladores de Microsoft, así como la amplia comunidad de ciencia de datos a fin de garantizar la estabilidad y la viabilidad general.

DSVM también incluye varios ejemplos de código y tutoriales en forma de blocs de notas de Jupyter que se pueden ejecutar localmente. También existe una documentación para estos ejemplos para facilitar la comprensión. 

Las herramientas, plataformas, utilidades y ejemplos incluidos en DSVM se han organizado en las siguientes secciones:

1. **[Marcos de aprendizaje profundo y de AI](dsvm-deep-learning-ai-frameworks.md)**
2. **[Herramientas de ciencia de datos y ML](dsvm-ml-data-science-tools.md)**
3. **[Lenguajes de programación](dsvm-languages.md)**
4. **[Herramientas de desarrollo e IDE](dsvm-tools-development.md)**
5. **[Herramientas de ingesta de datos](dsvm-tools-ingestion.md)**
6. **[Exploración de datos y herramientas de visualización](dsvm-tools-explore-and-visualize.md)**
7. **[Plataformas de datos](dsvm-data-platforms.md)**
8. **[Ejemplos y tutoriales](dsvm-samples-and-walkthroughs.md)**
