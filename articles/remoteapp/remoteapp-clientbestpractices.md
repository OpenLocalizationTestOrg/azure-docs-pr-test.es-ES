Hola siguiente información puede ayudarle a usar a Azure RemoteApp clientes:

* Utilice siempre el cliente más reciente de Hola. Esto garantiza que Hola versión del cliente está ejecutando tiene Hola últimas correcciones de errores, mejoras y características. Es posible que tenga toosign seguridad tooautomatically reciben las actualizaciones de cliente de Hola Hola de almacén adecuado.
* RemoteApp cerrará automáticamente la sesión si está inactivo durante un período de tiempo determinado. Orden tooprevent pérdida de datos, se recomienda cerrar las aplicaciones cuando haya terminado de utilizar el servicio de Hola.
