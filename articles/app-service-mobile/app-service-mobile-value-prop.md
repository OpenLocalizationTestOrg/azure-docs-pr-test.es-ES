## <a name="azure-app-service-features"></a>Características de Azure App Service
Hola después de características de la plataforma es útil para los sitios de producción móviles:

* **Escalado automático**: con el servicio de aplicación, rápidamente puede escalar verticalmente o escalar horizontalmente toohandle cualquier carga de cliente entrantes.
