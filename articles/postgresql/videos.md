# <a name="azure-database-for-postgresql-videos"></a>Vídeos de Azure Database for PostgreSQL
[Introducción al nuevo servicio de Azure Database for PostgreSQL](https://channel9.msdn.com/events/Build/2017/B8046)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8046/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

En este vídeo de la conferencia Microsoft //Build de 2017, obtenga información de dos clientes pioneros sobre cómo han usado el servicio de Azure Database for PostgreSQL para innovar más rápidamente. En el vídeo se ve cómo migraron al servicio y se analizan los pasos siguientes en el desarrollo de aplicaciones. Se describen algunas de las características clave del servicio y se analiza cómo los desarrolladores puedan migrar las aplicaciones existentes o desarrollar aplicaciones nuevas que usen este servicio administrado de PostgreSQL en Azure.
