# <a name="how-to-configure-an-availability-set-for-linux-virtual-machines-in-the-classic-deployment-model"></a>Configuración de un conjunto de disponibilidad para máquinas virtuales Linux en el modelo de implementación clásica
> [!IMPORTANT] 
> Azure tiene dos modelos de implementación diferentes para crear recursos y trabajar con ellos: [Resource Manager y el clásico](../../../resource-manager-deployment-model.md). En este artículo se trata el modelo de implementación clásico. Microsoft recomienda que las implementaciones más recientes usen el modelo del Administrador de recursos. También puede [configurar conjuntos de disponibilidad](../../azure-cli-arm-commands.md#azure-availset-commands-to-manage-your-availability-sets) en las implementaciones de Resource Manager.
