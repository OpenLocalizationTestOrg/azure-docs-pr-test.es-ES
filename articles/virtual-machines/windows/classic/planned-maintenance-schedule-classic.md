> [!IMPORTANT]
> Azure tiene dos modelos de implementación diferentes para crear recursos y trabajar con ellos: [Resource Manager y el clásico](../../../resource-manager-deployment-model.md). En este artículo se trata el modelo de implementación clásico. Microsoft recomienda que las implementaciones más recientes usen el modelo de Resource Manager. Para más información sobre el mantenimiento planeado en el modelo de Resource Manager, consulte [aquí](../planned-maintenance.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json).
