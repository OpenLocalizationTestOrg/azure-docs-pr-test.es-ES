# <a name="create-a-relay-namespace-using-hello-azure-portal"></a>Crear un espacio de nombres de retransmisión mediante Hola portal de Azure
Un espacio de nombres es un contenedor común para todos los componentes de Azure Relay. Varias instancias de Azure Relay pueden residir en un único espacio de nombres y los espacios de nombres suelen servir de contenedores de aplicación. No hay actualmente dos maneras diferentes toocreate un espacio de nombres de retransmisión.

1. Azure Portal (este artículo)
2. Plantillas de [Azure Resource Manager](../azure-resource-manager/resource-group-overview.md)
