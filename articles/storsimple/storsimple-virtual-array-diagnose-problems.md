# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Uso del servicio StorSimple Device Manager para solucionar problemas de StorSimple Virtual Array
## <a name="overview"></a>Información general

En la hoja de resumen del servicio, StorSimple Device Manager proporciona una opción **Diagnosticar y solucionar problemas** que resalta algunos de los problemas frecuentes que pueden producirse con la matriz virtual y cómo resolverlos. Este tutorial presenta la funcionalidad de autoservicio para solucionar problemas, proporcionada en el servicio StorSimple Device Manager.

![Diagnosticar y solucionar problemas](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnóstico y solución de problemas

Puede ver algunos de los problemas comunes con StorSimple Virtual Array y revisar sus soluciones directamente en la hoja de resumen del servicio StorSimple Device Manager.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Para diagnosticar un problema con la matriz virtual

1. Haga clic en **Diagnosticar y solucionar problemas**, en el panel izquierdo de la hoja de resumen del servicio StorSimple Device Manager, para ver una lista de problemas comunes.

2. **Expanda** el síntoma del problema que se ha encontrado para revisar los **Pasos recomendados** para ayudarle a solucionarlo. Si lo desea, también puede revisar los vínculos de documentación detallada proporcionados para más información.
   
    ![Solución de un problema](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Si no puede encontrar una referencia a su problema ni solucionarlo, acceda a Soporte técnico de Microsoft para obtener más ayuda.
