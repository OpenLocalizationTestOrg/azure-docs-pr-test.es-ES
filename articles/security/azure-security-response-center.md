# <a name="microsoft-security-response-center"></a>Centro de respuesta de seguridad de Microsoft
Algunos de los expertos en seguridad con más experiencia del mundo dirigen el Centro de respuestas de seguridad de Microsoft (MSRC). Estos expertos identifican, supervisan, responden y resuelven los incidentes de seguridad y las vulnerabilidades locales y en la nube a todas horas y todos los días del año.

Además del continuo trabajo, el equipo de MSRC tiene una serie de recursos disponibles para que pueda entender cómo proteger sus recursos e implementaciones de Azure de forma más eficaz.

## <a name="the-msrc-blog"></a>Blog del MSRC
El [blog de MSRC](https://blogs.technet.microsoft.com/msrc/) es el lugar para obtener las últimas noticias sobre lo que hace el MSRC para ayudarle a protegerse contra las amenazas en la nube.

## <a name="white-papers"></a>Notas del producto
MSRC ha publicado una serie de [notas del producto](https://technet.microsoft.com/library/bb969102.aspx) que le ayudan a comprender qué hacen y cómo lo hacen. Algunas proporcionan información sobre cómo se protege la nube de Microsoft e incluyen información útil sobre cómo puede emplear la misma configuración de seguridad.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Programas de interacción de investigadores de seguridad y recompensa
El MSRC admite la colaboración y las relaciones con investigadores de seguridad de todo el mundo para avanzar en la seguridad de los productos de Microsoft.

Los programas de recompensa de Microsoft pagan a los investigadores para:
- Técnicas de aprovechamiento novedosas
- Ideas defensivas que mitigan los aprovechamientos novedosos
- Identificación de vulnerabilidades críticas en software local y en la nube de Microsoft.

Para aprender más acerca de MSRC, visite la [página principal MSRC](https://technet.microsoft.com/library/dn440717.aspx).
