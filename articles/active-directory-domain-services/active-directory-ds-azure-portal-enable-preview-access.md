# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Habilitación de la experiencia de versión preliminar para administrar el dominio administrado desde Azure Portal

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Su dominio administrado existente no se muestra en la experiencia de Azure Portal (versión preliminar)
Tiene un dominio administrado de [Azure AD Domain Services](active-directory-ds-overview.md) que se ha habilitado mediante el Portal de Azure clásico. Cuando vaya a Azure Portal, la hoja de Azure AD Domain Services (versión preliminar) no mostrará este dominio administrado existente.

![El dominio administrado existente no se muestra en Azure Portal](./media/getting-started/missing-domain-services-instance.png)

Esto se debe a que el dominio administrado existente no se ha migrado al nuevo Azure Portal. Puede ponerse en contacto con nosotros para habilitar la nueva experiencia de Azure Portal (versión preliminar) para su dominio administrado existente.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Use la versión preliminar a fin de probar la experiencia de Azure Portal para Azure AD Domain Services
Siga las instrucciones de la página [Póngase en contacto con nosotros](active-directory-ds-contact-us.md) y envíenos un mensaje de correo electrónico. Habilitaremos enseguida la versión preliminar de Azure Portal para su dominio administrado existente.

> [!NOTE]
> Cuando la nueva experiencia de Azure Portal para Azure AD Domain Services esté disponible con carácter general (GA), todos los dominios administrados existentes se migrarán de modo automático. En ese momento, podrá administrar el dominio administrado existente desde Azure Portal.
