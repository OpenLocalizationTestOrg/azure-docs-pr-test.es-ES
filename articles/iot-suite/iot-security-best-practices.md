## <a name="iot-hardware-manufacturerintegrator"></a><span data-ttu-id="adb78-119">Integrador/fabricante de hardware IoT</span><span class="sxs-lookup"><span data-stu-id="adb78-119">IoT hardware manufacturer/integrator</span></span>
<span data-ttu-id="adb78-120">Estos son los procedimientos recomendados para los fabricantes de hardware de IoT e integradores de hardware.</span><span class="sxs-lookup"><span data-stu-id="adb78-120">The following are the best practices for IoT hardware manufacturers and hardware integrators.</span></span>

* <span data-ttu-id="adb78-121">**Acotar el hardware a unos requisitos mínimos**: el diseño del hardware debe incluir únicamente las características mínimas necesarias para su funcionamiento.</span><span class="sxs-lookup"><span data-stu-id="adb78-121">**Scope hardware to minimum requirements**: The hardware design should include the minimum features required for operation of the hardware, and nothing more.</span></span> <span data-ttu-id="adb78-122">Un ejemplo es la inclusión de puertos USB únicamente si es necesario para el funcionamiento del dispositivo.</span><span class="sxs-lookup"><span data-stu-id="adb78-122">An example is to include USB ports only if necessary for the operation of the device.</span></span> <span data-ttu-id="adb78-123">Estas características adicionales abren el dispositivo para vectores de ataques no deseados, los cuales deben evitarse.</span><span class="sxs-lookup"><span data-stu-id="adb78-123">These additional features open the device for unwanted attack vectors that should be avoided.</span></span>
* <span data-ttu-id="adb78-124">**Crear hardware a prueba de manipulaciones**: mecanismos integrados para detectar la alteración física del hardware; por ejemplo, abrir la cubierta del dispositivo o quitar una parte de este.</span><span class="sxs-lookup"><span data-stu-id="adb78-124">**Make hardware tamper proof**: Build in mechanisms to detect physical tampering, such as opening of the device cover or removing a part of the device.</span></span> <span data-ttu-id="adb78-125">Estas señales de alteración pueden formar parte de la transmisión de datos que se carga en la nube y que podría alertar a los operadores sobre estos eventos.</span><span class="sxs-lookup"><span data-stu-id="adb78-125">These tamper signals may be part of the data stream uploaded to the cloud, which could alert operators of these events.</span></span>
